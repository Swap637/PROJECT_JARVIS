Menu Shortcuts
Ctrl + 0 Opens transaction screen.
Ctrl + 1 Opens transaction passing screen.
Ctrl + 2 Opens RD/FD Recknor screen.
Ctrl + 3 Opens EMI Recknor screen.
Ctrl + 5 Opens Ledger view screen.
Ctrl + 8 Opens Client Ledger screen.
Alt + L Opens Ledger view for specific account. (from Transaction entry screen.)
Ctrl + Sft + R Opens Reports Selector screen.
Ctrl + Sft + S Opens Account Search screen.


Action Shortcuts
Ctrl + S Saves the entry/transaction.
Ctrl + U Updates the entry/transaction.
Ctrl + D Deletes the entry/transaction.
Ctrl + J Opens the search screen.
Ctrl + K Resets the screen to initial state.
Ctrl + Sft + Q Logs out.
